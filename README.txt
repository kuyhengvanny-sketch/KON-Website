# KON Website — How to Use

## Files
- index.html  — all the page content (text, structure)
- style.css   — all colors, fonts, spacing
- script.js   — mobile menu, animations, lightbox
- images/     — put all your photos here

## How to add your photos

Create a folder called "images" in the same place as these files.
Add your photos with these exact names:

HERO
  images/hero-photo.jpg         — photo of a mother or consultation

STORY
  images/story-main.jpg         — team photo or Kampong Cham survey
  images/story-2.jpg            — community or research photo
  images/story-3.jpg            — team working photo

TEAM (square photos work best)
  images/team-vanny.jpg
  images/team-molinna.jpg
  images/team-layhok.jpg
  images/team-hanarita.jpg
  images/team-seangleng.jpg
  images/team-panhasith.jpg

GALLERY
  images/gallery-1.jpg          — large featured photo
  images/gallery-2.jpg
  images/gallery-3.jpg
  images/gallery-4.jpg
  images/gallery-5.jpg

If a photo is missing the site still works — it shows a purple placeholder.

## How to edit text

Open index.html in any text editor (Notepad, VS Code).
Search for the text you want to change and type your new text.
Save the file.

## Two links to update before launching

1. Find this line in index.html:
   href="https://forms.google.com"
   Replace with your actual Google Form booking link.

2. Find this line:
   href="https://t.me/KONHealth"
   Replace with your actual Telegram username.

## How to change colors

Open style.css.
At the very top you will see :root { ... }
Change the hex color codes there.
The main brand color is --purple-600: #534AB7

## How to upload to konhealth.org

Upload ALL files to your Namecheap hosting:
- index.html
- style.css
- script.js
- images/ folder with all photos

Use Namecheap's File Manager in cPanel.
